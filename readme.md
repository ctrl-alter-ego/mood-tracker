# Mood Tracker (standalone)

This folder contains a standalone version of the Mood Tracker that was previously bundled into the training plan generator site.

## Local use

Open `mood-tracker/index.html` directly in a browser, or serve the repo with any static file server.

## Deploy as a separate Netlify site

Create a new Netlify site from this repository and set:

- Base directory: `mood-tracker`
- Build command: (leave empty)
- Publish directory: `.`
